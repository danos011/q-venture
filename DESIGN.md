# DESIGN — Q Venture Capital

## Тема

Тёмная — осознанно: кинематографичный лендинг с фоновым видео (сине-фиолетовые
«лава-лампа» блобы поверх near-black). Смотрят вечером с десктопа/телефона,
первое касание — интро-видео с кроссфейдом.

## Палитра

- `--color-q-black: #050505` — фон (body), поверх — видео + виньетка.
- `--color-q-purple: #7e3aae` — единственный цветовой акцент: подзаголовки, термины, ховеры.
- `--color-q-indigo: #32327e` — только как полупрозрачная подложка карточек (`/15`–`/25`).
- Белый в тексте всегда с альфой: 100% заголовки, 85% тело, 70% лиды, 40–50% сервисный текст.

## Типографика

- **BigStem** (`font-bigstem`) — display: гигантские UPPERCASE заголовки, узкий гротеск.
  Названия секций на латинице, контентные заголовки на русском.
- **Bajazzo** (`font-bajazzo`) — тело: статьи, описания, сервисные подписи
  (в подписях — tracking 0.3–0.5em + uppercase).
- Статьи: колонка ≤72ch, text-lg, leading 1.7–1.8, обычный регистр.

## Композиция

- Секции: `px-5 md:px-8 lg:px-12`, вертикальный ритм `gap-[10vh]` между секциями.
- Асимметрия: диагональ hero, стаггер колонок «Сфер» (`md:mt-16/0/32`).
- Призрачные wordmarks: класс `.ghost-word` (текст с обводкой white/12, clamp 8–20rem), только lg+.
- Списки Портфолио/Контактов: редакционные строки с `border-white/10`, ховер — фон indigo/20 + сдвиг.
- Карточки только для Базы знаний (сетка) и «Кто мы?» — не плодить.

## Мошн

- Интро → контент: кроссфейд 1s; reveal-хореография через `[data-intro="done"]` + delays.
- Скролл: `.scroll-reveal` (opacity + translateY 28px, cubic-bezier(0.22,1,0.36,1), 0.8s), стаггер ≤0.15s.
- Всё уважает `prefers-reduced-motion`.
- Иконки сфер: статичный webp → анимированный webp с альфой при входе в вьюпорт
  (НЕ конвертировать в видео — yuv420p убивает прозрачность).
